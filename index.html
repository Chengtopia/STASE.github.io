Overview

STASE introduces a novel framework for generating spatially dynamic musical compositions directly from natural language prompts. Unlike existing text-to-audio systems that produce monophonic or fixed-stereo outputs, STASE offers fine-grained control over the spatialization of individual sound sources through a hybrid neuro-symbolic architecture.

Method

Overall Method

  1. Input preparation: natural-language instruction and optional references.
  2. Prompt structuring: convert input into description and abstract prompts to control modules.
  3. Spatial knowledge retrieval (RAG): fetch prior cues (e.g., room, direction, HRIR/BRIR) and map them to constraints.
  4. Source processing: separate or manage stems; estimate spatial attributes such as direction, distance and reverberation.
  5. Rendering and mixing: spatialize each stem with consistent acoustics and mix them into a coherent scene.
  6. Output: deliver the final spatial audio mix.

The overall workflow of the STASE framework. Natural language input is progressively structured into description and abstract prompts, enriched by retrieval-augmented spatial knowledge (RAG). These prompts steer modules for source extraction, spatial attribute estimation (e.g., direction, distance, room cues), and rendering. The pipeline finally mixes all rendered stems to produce a coherent spatial audio result.

Concretely, STASE couples: (1) language understanding to form controllable prompts; (2) a spatial information matcher that aligns text with symbolic/learned priors; (3) a render-and-mix stage that positions sources in space with consistent acoustics.
